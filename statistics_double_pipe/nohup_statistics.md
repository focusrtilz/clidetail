## Before

__The commands before `nohup`:__ If the given command failed, execute `nohup`.

| Command | percentage |
|--------|--------|
| grep | 16% |
| ps | 16% |
| ssh | 16% |
| awk | 16% |
| w | 16% |
| ss | 16% |



## After

__The commands after `nohup`:__ If `nohup` failed, execute the given command.
