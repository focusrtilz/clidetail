## Before

__The commands before `history`:__ If the given command failed, execute `history`.

| Command | percentage |
|--------|--------|
| rm | 20% |
| echo | 20% |
| alias | 20% |
| w | 20% |
| nc | 20% |



## After

__The commands after `history`:__ If `history` failed, execute the given command.

| Command | Percentage | 
|-------|--------|
| head | 33% |
| sort | 33% |
| ln | 33% |
