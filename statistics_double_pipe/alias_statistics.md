## Before

__The commands before `alias`:__ If the given command failed, execute `alias`.

| Command | percentage |
|--------|--------|
| cd | 20% |
| pwd | 20% |
| alias | 20% |
| w | 20% |
| nc | 20% |



## After

__The commands after `alias`:__ If `alias` failed, execute the given command.

| Command | Percentage | 
|-------|--------|
| echo | 33% |
| grep | 11% |
| cd | 11% |
| alias | 11% |
| tail | 11% |
| history | 11% |
| sed | 11% |
