## Before

__The commands before `ps`:__ If the given command failed, execute `ps`.

| Command | percentage |
|--------|--------|
| name | 33% |
| echo | 11% |
| id | 11% |
| hostname | 11% |
| host | 11% |
| ip | 11% |
| ln | 11% |



## After

__The commands after `ps`:__ If `ps` failed, execute the given command.

| Command | Percentage | 
|-------|--------|
| name | 27% |
| echo | 18% |
| ssh | 9% |
| cal | 9% |
| ss | 9% |
| nohup | 9% |
| host | 9% |
| ln | 9% |
