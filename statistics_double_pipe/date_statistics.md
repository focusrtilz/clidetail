## Before

__The commands before `date`:__ If the given command failed, execute `date`.

| Command | percentage |
|--------|--------|
| w | 21% |
| date | 15% |
| id | 10% |
| grep | 5% |
| cd | 5% |
| wget | 5% |
| diff | 5% |
| name | 5% |
| ping | 5% |
| echo | 5% |
| curl | 5% |
| su | 5% |
| ss | 5% |



## After

__The commands after `date`:__ If `date` failed, execute the given command.

| Command | Percentage | 
|-------|--------|
| echo | 50% |
| date | 21% |
| id | 21% |
| man | 7% |
