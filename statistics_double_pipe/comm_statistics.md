## Before

__The commands before `comm`:__ If the given command failed, execute `comm`.

| Command | percentage |
|--------|--------|
| man | 15% |
| comm | 15% |
| echo | 15% |
| free | 7% |
| awk | 7% |
| w | 7% |
| ssh | 7% |
| screen | 7% |
| exit | 7% |
| ss | 7% |



## After

__The commands after `comm`:__ If `comm` failed, execute the given command.

| Command | Percentage | 
|-------|--------|
| echo | 18% |
| man | 18% |
| comm | 18% |
| sudo | 9% |
| su | 9% |
| ssh | 9% |
| ss | 9% |
| exit | 9% |
