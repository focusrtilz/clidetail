## Before

__The commands before `awk`:__ If the given command failed, execute `awk`.

| Command | percentage |
|--------|--------|
| free | 25% |
| awk | 25% |
| w | 25% |
| ss | 25% |



## After

__The commands after `awk`:__ If `awk` failed, execute the given command.
