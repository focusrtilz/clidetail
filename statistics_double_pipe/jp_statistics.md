## Before

__The commands before `jp`:__ If the given command failed, execute `jp`.

| Command | percentage |
|--------|--------|
| jp | 40% |
| curl | 20% |
| ping | 20% |
| w | 20% |



## After

__The commands after `jp`:__ If `jp` failed, execute the given command.

| Command | Percentage | 
|-------|--------|
| echo | 42% |
| ping | 28% |
| jp | 28% |
