## Before

__The commands before `exit`:__ If the given command failed, execute `exit`.

| Command | percentage |
|--------|--------|
| sed | 20% |
| cd | 10% |
| cal | 10% |
| nc | 10% |
| man | 10% |
| comm | 10% |
| echo | 10% |
| wget | 10% |
| w | 10% |



## After

__The commands after `exit`:__ If `exit` failed, execute the given command.

| Command | Percentage | 
|-------|--------|
| man | 25% |
| ssh | 25% |
| ss | 25% |
| comm | 25% |
