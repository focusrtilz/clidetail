## Before

__The commands before `free`:__ If the given command failed, execute `free`.

## After

__The commands after `free`:__ If `free` failed, execute the given command.

| Command | Percentage | 
|-------|--------|
| grep | 8% |
| cp | 8% |
| man | 8% |
| echo | 8% |
| sed | 8% |
| awk | 8% |
| w | 8% |
| uptime | 8% |
| ss | 8% |
| sleep | 8% |
| comm | 8% |
| bc | 8% |
