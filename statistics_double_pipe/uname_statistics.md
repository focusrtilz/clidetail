## Before

__The commands before `uname`:__ If the given command failed, execute `uname`.

| Command | percentage |
|--------|--------|
| name | 29% |
| uname | 23% |
| ls | 11% |
| cat | 11% |
| echo | 5% |
| id | 5% |
| hostname | 5% |
| host | 5% |



## After

__The commands after `uname`:__ If `uname` failed, execute the given command.

| Command | Percentage | 
|-------|--------|
| uname | 33% |
| name | 33% |
| cat | 16% |
| rm | 16% |
