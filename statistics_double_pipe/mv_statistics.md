## Before

__The commands before `mv`:__ If the given command failed, execute `mv`.

| Command | percentage |
|--------|--------|
| w | 22% |
| cat | 11% |
| ls | 11% |
| echo | 11% |
| name | 11% |
| exec | 11% |
| mv | 11% |
| ss | 11% |



## After

__The commands after `mv`:__ If `mv` failed, execute the given command.

| Command | Percentage | 
|-------|--------|
| mv | 25% |
| ls | 25% |
| echo | 25% |
| ss | 25% |
