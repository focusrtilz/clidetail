## Before

__The commands before `whois`:__ If the given command failed, execute `whois`.

| Command | percentage |
|--------|--------|
| ss | 100% |



## After

__The commands after `whois`:__ If `whois` failed, execute the given command.

| Command | Percentage | 
|-------|--------|
| sudo | 33% |
| su | 33% |
| ip | 33% |
