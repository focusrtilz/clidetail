## Before

__The commands before `uptime`:__ If the given command failed, execute `uptime`.

| Command | percentage |
|--------|--------|
| echo | 12% |
| name | 12% |
| id | 12% |
| hostname | 12% |
| host | 12% |
| free | 12% |
| awk | 12% |
| w | 12% |



## After

__The commands after `uptime`:__ If `uptime` failed, execute the given command.
