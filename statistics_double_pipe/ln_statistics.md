## Before

__The commands before `ln`:__ If the given command failed, execute `ln`.

| Command | percentage |
|--------|--------|
| cal | 25% |
| ps | 12% |
| nc | 12% |
| host | 12% |
| history | 12% |
| ping | 12% |
| w | 12% |



## After

__The commands after `ln`:__ If `ln` failed, execute the given command.

| Command | Percentage |
