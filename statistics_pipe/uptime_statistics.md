## Before

__The commands before `uptime`:__ Pipe the result of the given command to `uptime`.

| Command | percentage |
|--------|--------|
| sed | 25% |
| w | 25% |
| echo | 25% |
| tail | 25% |



## After

__The commands after `uptime`:__ Pipe the result of `uptime` to the given command.
