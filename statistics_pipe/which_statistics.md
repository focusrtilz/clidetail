## Before

__The commands before `which`:__ Pipe the result of the given command to `which`.

| Command | percentage |
|--------|--------|
| which | 23% |
| w | 23% |
| nc | 15% |
| su | 7% |
| man | 7% |
| alias | 7% |

## After

__The commands after `which`:__ Pipe the result of `which` to the given command.
