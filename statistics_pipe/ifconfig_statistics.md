## Before

__The commands before `ifconfig`:__ Pipe the result of the given command to `ifconfig`.

| Command | percentage |
|--------|--------|
| echo | 20% |
| ifconfig | 20% |
| sort | 13% |
| head | 6% |
| cat | 6% |
| ip | 6% |
| name | 6% |
| grep | 6% |
| awk | 6% |
| w | 6% |
