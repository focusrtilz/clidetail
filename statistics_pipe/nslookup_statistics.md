## Before

__The commands before `nslookup`:__ Pipe the result of the given command to `nslookup`.

| Command | percentage |
|--------|--------|
| w | 25% |
| name | 25% |
| hostname | 25% |
| host | 25% |



## After

__The commands after `nslookup`:__ Pipe the result of `nslookup` to the given command.

| Command | Percentage | 
|-------|--------|
| grep | 31% |
| dd | 12% |
| ss | 12% |
| sed | 12% |
| name | 12% |
| tail | 6% |
