## Before

__The commands before `whoami`:__ Pipe the result of the given command to `whoami`.

| Command | percentage |
|--------|--------|
| ps | 33% |
| finger | 33% |
| last | 33% |



## After

__The commands after `whoami`:__ Pipe the result of `whoami` to the given command.

| Command | Percentage | 
|-------|--------|
| grep | 33% |
| awk | 22% |
| w | 22% |
| head | 11% |
| echo | 11% |
