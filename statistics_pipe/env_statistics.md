## Before

__The commands before `env`:__ Pipe the result of the given command to `env`.

| Command | percentage |
|--------|--------|
| cd | 25% |
| ps | 25% |
| id | 25% |
| cat | 25% |



## After

__The commands after `env`:__ Pipe the result of `env` to the given command.

| Command | Percentage | 
|-------|--------|
| grep | 33% |
| head | 22% |
| sort | 11% |
| ps | 11% |
| sed | 11% |
| echo | 11% |
