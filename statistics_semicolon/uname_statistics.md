## Before

__The commands before `uname`:__  Execute the given command __before__ `uname`.

| Command | percentage |
|--------|--------|
| sed | 20% |
| uname | 14% |
| name | 14% |
| echo | 8% |
| grep | 5% |
| awk | 5% |
| w | 5% |
| mysql | 5% |
| cat | 2% |
| host | 2% |
| clear | 2% |
| export | 2% |
| sudo | 2% |
| su | 2% |



## After

__The commands after `uname`:__ Execute the given command __after__ `uname`.

| Command | Percentage | 
|-------|--------|
| uname | 31% |
| name | 31% |
| echo | 12% |
| mysql | 12% |
| grep | 6% |
| mv | 6% |
