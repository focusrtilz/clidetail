## Before

__The commands before `cmp`:__  Execute the given command __before__ `cmp`.

| Command | percentage |
|--------|--------|
| cp | 16% |
| du | 16% |
| sed | 16% |
| w | 16% |
| echo | 16% |
| ip | 16% |



## After

__The commands after `cmp`:__ Execute the given command __after__ `cmp`.

| Command | Percentage | 
|-------|--------|
| echo | 60% |
| dd | 20% |
| ip | 20% |
