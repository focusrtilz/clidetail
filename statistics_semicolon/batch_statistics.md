## Before

__The commands before `batch`:__  Execute the given command __before__ `batch`.

| Command | percentage |
|--------|--------|
| id | 18% |
| vi | 9% |
| tac | 9% |
| grep | 9% |
| echo | 9% |
| ps | 9% |
| du | 9% |
| awk | 9% |
| xargs | 9% |
| w | 9% |



## After

__The commands after `batch`:__ Execute the given command __after__ `batch`.

| Command | Percentage | 
|-------|--------|
| sudo | 22% |
| su | 22% |
| reboot | 22% |
| grep | 11% |
| head | 11% |
| du | 11% |
