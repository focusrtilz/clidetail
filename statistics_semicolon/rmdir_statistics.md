## Before

__The commands before `rmdir`:__  Execute the given command __before__ `rmdir`.

| Command | percentage |
|--------|--------|
| find | 16% |
| rsync | 8% |
| nc | 8% |
| cd | 8% |
| grep | 8% |
| w | 8% |
| mv | 8% |
| xargs | 8% |
| name | 8% |
| mkdir | 8% |
| alias | 8% |



## After

__The commands after `rmdir`:__ Execute the given command __after__ `rmdir`.

| Command | Percentage | 
|-------|--------|
| mkdir | 50% |
| cd | 50% |
