## Before

__The commands before `free`:__  Execute the given command __before__ `free`.

| Command | percentage |
|--------|--------|
| echo | 15% |
| df | 15% |
| w | 15% |
| cat | 7% |
| grep | 7% |
| cp | 7% |
| tail | 7% |
| ss | 7% |
| name | 7% |
| alias | 7% |
