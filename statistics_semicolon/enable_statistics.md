## Before

__The commands before `enable`:__  Execute the given command __before__ `enable`.

| Command | percentage |
|--------|--------|
| w | 25% |
| name | 25% |
| grep | 12% |
| yes | 12% |
| awk | 12% |
| sed | 12% |



## After

__The commands after `enable`:__ Execute the given command __after__ `enable`.

| Command | Percentage | 
|-------|--------|
| ls | 40% |
| echo | 20% |
| tar | 20% |
| sleep | 20% |
