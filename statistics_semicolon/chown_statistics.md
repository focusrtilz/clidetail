## Before

__The commands before `chown`:__  Execute the given command __before__ `chown`.

| Command | percentage |
|--------|--------|
| w | 30% |
| ls | 10% |
| grep | 10% |
| chmod | 10% |
| find | 10% |
| echo | 10% |
| sed | 10% |
| ip | 10% |



## After

__The commands after `chown`:__ Execute the given command __after__ `chown`.

| Command | Percentage | 
|-------|--------|
| sudo | 25% |
| tar | 25% |
| su | 25% |
| nc | 25% |
