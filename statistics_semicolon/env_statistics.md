## Before

__The commands before `env`:__  Execute the given command __before__ `env`.

| Command | percentage |
|--------|--------|

| vi | 23% |
| ip | 7% |

| grep | 7% |
| ss | 7% |

## After

__The commands after `env`:__ Execute the given command __after__ `env`.

| Command | Percentage |
