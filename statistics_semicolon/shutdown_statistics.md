## Before

__The commands before `shutdown`:__  Execute the given command __before__ `shutdown`.

| Command | percentage |
|--------|--------|
| sleep | 100% |



## After

__The commands after `shutdown`:__ Execute the given command __after__ `shutdown`.

| Command | Percentage | 
|-------|--------|
| sleep | 100% |
