## Before

__The commands before `rsync`:__  Execute the given command __before__ `rsync`.

| Command | percentage |
|--------|--------|
| w | 16% |
| id | 16% |
| cd | 16% |
| echo | 16% |
| mount | 16% |
| date | 16% |



## After

__The commands after `rsync`:__ Execute the given command __after__ `rsync`.

| Command | Percentage | 
|-------|--------|
| echo | 33% |
| rm | 11% |
| rmdir | 11% |
| man | 11% |
| nc | 11% |
| cd | 11% |
| mkdir | 11% |
