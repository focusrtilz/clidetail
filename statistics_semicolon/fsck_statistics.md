## Before

__The commands before `fsck`:__  Execute the given command __before__ `fsck`.

| Command | percentage |
|--------|--------|
| mount | 100% |



## After

__The commands after `fsck`:__ Execute the given command __after__ `fsck`.

| Command | Percentage | 
|-------|--------|
| mount | 33% |
| w | 33% |
| vi | 33% |
