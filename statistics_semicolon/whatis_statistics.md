## Before

__The commands before `whatis`:__  Execute the given command __before__ `whatis`.

| Command | percentage |
|--------|--------|
| cp | 12% |
| whatis | 12% |
| ip | 12% |
| w | 12% |
| exec | 12% |
| ls | 12% |
| man | 12% |
| echo | 12% |



## After

__The commands after `whatis`:__ Execute the given command __after__ `whatis`.

| Command | Percentage | 
|-------|--------|
| w | 16% |
| echo | 16% |
| curl | 8% |
| ss | 8% |
| date | 8% |
| whatis | 8% |
| ip | 8% |
| cat | 8% |
| more | 8% |
| vi | 8% |
