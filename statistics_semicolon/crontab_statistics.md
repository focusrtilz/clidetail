## Before

__The commands before `crontab`:__  Execute the given command __before__ `crontab`.

| Command | percentage |
|--------|--------|
| cron | 23% |
| echo | 23% |
| dmesg | 7% |
| ls | 7% |
| passwd | 7% |
| w | 7% |
| ss | 7% |
| sed | 7% |
| crontab | 7% |



## After

__The commands after `crontab`:__ Execute the given command __after__ `crontab`.

| Command | Percentage | 
|-------|--------|
| echo | 25% |
| cat | 12% |
| dmesg | 12% |
| man | 12% |
| cron | 12% |
| crontab | 12% |
| comm | 12% |
