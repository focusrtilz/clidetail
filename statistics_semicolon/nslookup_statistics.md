## Before

__The commands before `nslookup`:__  Execute the given command __before__ `nslookup`.

| Command | percentage |
|--------|--------|
| echo | 22% |
| awk | 11% |
| w | 11% |
| name | 11% |
| hostname | 11% |
| host | 11% |
| cat | 11% |
| xargs | 11% |



## After
