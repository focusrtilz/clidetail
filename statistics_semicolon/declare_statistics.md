## Before

__The commands before `declare`:__  Execute the given command __before__ `declare`.

| Command | percentage |
|--------|--------|
| alias | 25% |
| nc | 16% |
| name | 8% |
| date | 8% |
| man | 8% |
| which | 8% |
| w | 8% |
| comm | 8% |
| unalias | 8% |



## After

__The commands after `declare`:__ Execute the given command __after__ `declare`.

| Command | Percentage | 
|-------|--------|
| echo | 22% |
| nc | 22% |
| man | 11% |
| alias | 11% |
| which | 11% |
| w | 11% |
| comm | 11% |
