## Before

__The commands before `useradd`:__  Execute the given command __before__ `useradd`.

| Command | percentage |
|--------|--------|
| mkdir | 50% |
| name | 50% |



## After

__The commands after `useradd`:__ Execute the given command __after__ `useradd`.

| Command | Percentage | 
|-------|--------|
| passwd | 22% |
| w | 22% |
| ss | 22% |
| echo | 11% |
| password | 11% |
| name | 11% |
