## Before

__The commands before `dmesg`:__  Execute the given command __before__ `dmesg`.

| Command | percentage |
|--------|--------|
| cron | 25% |
| crontab | 25% |
| ls | 25% |
| grep | 25% |



## After

__The commands after `dmesg`:__ Execute the given command __after__ `dmesg`.

| Command | Percentage | 
|-------|--------|
| cron | 50% |
| crontab | 50% |
