## Before

__The commands before `whois`:__  Execute the given command __before__ `whois`.

| Command | percentage |
|--------|--------|
| echo | 25% |
| w | 25% |
| whois | 6% |
| netstat | 6% |
| awk | 6% |
| grep | 6% |
| ps | 6% |
| wget | 6% |
| sed | 6% |
| ss | 6% |



## After

__The commands after `whois`:__ Execute the given command __after__ `whois`.

| Command | Percentage | 
|-------|--------|
| echo | 31% |
| sed | 12% |
| ip | 12% |
| grep | 6% |
| df | 6% |
| whois | 6% |
| w | 6% |
| dd | 6% |
| sudo | 6% |
| su | 6% |
