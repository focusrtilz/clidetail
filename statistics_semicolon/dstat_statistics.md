## Before

__The commands before `dstat`:__  Execute the given command __before__ `dstat`.

| Command | percentage |
|--------|--------|
| grep | 12% |
| cp | 12% |
| ps | 12% |
| head | 12% |
| sort | 12% |
| awk | 12% |
| w | 12% |
| id | 12% |



## After

__The commands after `dstat`:__ Execute the given command __after__ `dstat`.

| Command | Percentage | 
|-------|--------|
| screen | 100% |
