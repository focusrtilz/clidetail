## Before

__The commands before `cpio`:__  Execute the given command __before__ `cpio`.

| Command | percentage |
|--------|--------|
| cd | 60% |
| cp | 20% |
| cpio | 20% |



## After

__The commands after `cpio`:__ Execute the given command __after__ `cpio`.

| Command | Percentage | 
|-------|--------|
| cd | 25% |
| cp | 25% |
| rm | 25% |
| cpio | 25% |
